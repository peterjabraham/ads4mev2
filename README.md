# Ad Generator App

An AI-powered ad generation tool built with Next.js, TypeScript, and OpenAI.

## Features

- AI-powered ad generation
- Campaign management
- Template system
- History tracking
- Saved headlines
- Real-time preview

## Tech Stack

- Next.js 13+ (App Router)
- TypeScript
- Zustand for state management
- Material-UI
- Tailwind CSS
- OpenAI API
